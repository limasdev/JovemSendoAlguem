# shared/components/layout/Layout.tsx

## Utilidade
Componente de layout principal que envolve todas as páginas da aplicação.

## Funcionalidades
- **Header/Navbar**: Barra de navegação superior com logo e menu
- **Tema**: Botão para alternar entre tema claro e escuro
- **Navegação Desktop**: Links para Visão Geral e Fluxo de Caixa
- **Navegação Mobile**: 
  - Navbar inferior com ícones
  - Menu hambúrguer que abre drawer lateral
- **Drawer**: Painel lateral com configurações (tema, notificações, perfil)
- **Responsividade**: Adapta-se automaticamente a desktop e mobile

## Componentes Internos
- Logo com navegação para home
- Links de navegação com indicador de ativo
- Botão de tema (sol/lua)
- Notificações com badge
- Avatar do usuário
- Menu mobile (hambúrguer)
- Drawer de configurações

## Importância
É o componente base que define a estrutura visual de todas as páginas.

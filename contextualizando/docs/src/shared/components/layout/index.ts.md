# shared/components/layout/index.ts

## Utilidade
Arquivo de exportação do componente Layout.

## Funcionalidades
- Exporta o componente Layout para uso em outros módulos
- Centraliza acesso ao componente de layout

## Exportações
- `Layout`: Componente principal de layout da aplicação

## Uso
```typescript
import { Layout } from './shared/components/layout';
```

## Importância
Permite importar o Layout sem conhecer a estrutura interna da pasta.

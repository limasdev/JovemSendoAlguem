# app/routes.tsx

## Utilidade
Arquivo central de configuração das rotas da aplicação usando React Router.

## Funcionalidades
- Define todas as rotas da aplicação
- Configura o Layout como componente base (rota pai)
- Mapeia URLs para componentes específicos
- Suporte a navegação entre Dashboard e Fluxo de Caixa

## Rotas Configuradas
- `/` → Dashboard (página inicial)
- `/fluxo-caixa` → Fluxo de Caixa

## Importância
Este arquivo é o ponto central de navegação da aplicação. Qualquer nova página deve ser registrada aqui.

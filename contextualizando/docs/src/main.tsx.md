# main.tsx

## Utilidade
Ponto de entrada (entry point) da aplicação React.

## Funcionalidades
- Inicializa o React com StrictMode
- Cria o root element no DOM
- Renderiza o componente AppRoutes
- Importa estilos globais (index.css)

## Componentes
- `StrictMode`: Modo estrito do React para detectar problemas
- `AppRoutes`: Componente de rotas da aplicação

## Fluxo
1. Busca o elemento #root no HTML
2. Cria a raiz do React
3. Renderiza as rotas dentro do StrictMode

## Importância
Arquivo que inicia toda a aplicação. É o primeiro código executado.

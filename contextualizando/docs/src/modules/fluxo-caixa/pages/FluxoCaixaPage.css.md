# modules/fluxo-caixa/pages/FluxoCaixaPage.css

## Utilidade
Folha de estilos CSS da página Fluxo de Caixa.

## Funcionalidades
- **Layout em Cards**: Grid com 3 cards (Cadastro, Resumo, Histórico)
- **Card de Cadastro**: Formulário com campos organizados
- **Card de Resumo**: Cards de totais com cores (verde/vermelho)
- **Card de Histórico**: Tabela de transações com scroll horizontal
- **Toggle de Tipo**: Botões estilizados para Entrada/Saída
- **Tabela**: Cabeçalho fixo, linhas com hover, cores diferenciadas
- **Badges**: Categorias visuais para Tipo, Status e Pagamento
- **Inputs**: Estilização de campos com foco verde
- **Botões**: Primário (verde) e Ghost (subtle)
- **Efeito Ambiente**: Glow animado no background
- **Responsividade**: Adaptação para mobile (cards empilhados)

## Classes Principais
- `.fx-cards-grid`: Grid principal dos cards
- `.fx-card-cadastro`, `.fx-card-resumo`, `.fx-card-historico`: Cards específicos
- `.fx-type-toggle`: Container dos botões Entrada/Saída
- `.fx-table`: Tabela de histórico
- `.fx-status-badge`, `.fx-type-badge`, `.fx-payment-badge`: Badges visuais
- `.fx-resumo-card`: Cards de resumo com cores

## Design System
Utiliza variáveis CSS para cores, mantendo consistência com o tema global.
